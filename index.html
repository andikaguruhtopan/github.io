My Portfolio

Project 1

Dual-WAN Enterprise Network with VLAN Segmentation & Hotspot Management

Designed and implemented an enterprise-style MikroTik network featuring dual-WAN connectivity, VLAN segmentation for Finance, HR, and Staff departments, hotspot authentication, and profile-based bandwidth management.

Configured Dual-WAN routing using Policy-Based Routing (PBR) with dedicated routing tables for each ISP. Static and backup routes ensure traffic distribution and automatic failover when one WAN connection becomes unavailable.

Implemented application-based traffic steering using MikroTik Mangle and Routing Marks. General traffic from the Management network is routed through ISP 2, while latency-sensitive gaming traffic is automatically redirected to ISP 1 to achieve lower latency and improved network performance.

Implemented automatic WAN failover using Netwatch and static backup routes. When ISP 2 becomes unavailable, traffic is seamlessly redirected to ISP 1 while Telegram Bot instantly sends outage notifications, ensuring uninterrupted network connectivity.
